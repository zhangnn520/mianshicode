# 18 - 二叉树的镜像

## 题目描述
操作给定的二叉树，将其变换为源二叉树的镜像。

## 题解
递归依次交换左右子节点。
更优题解，参考leetcode 226.

```python
# -*- coding:utf-8 -*-
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None
class Solution:
    # 返回镜像树的根节点
    def Mirror(self, root):
        # write code here
        if not root: return
 
        self.SwapNode(root)
        self.Mirror(root.left)
        self.Mirror(root.right)
 
    def SwapNode(self, root):
        temp = root.left
        root.left = root.right
        root.right = temp
```
